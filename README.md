## Getting started

- clone this repository
- open in your favorite code editor
- create .env file inside root folder
```
DEEPSEEK_API_KEY=<deepseek-apiLey>

```
- install dependencies `npm install`
- build `npx tsc`
- start app `node dist/main.js`
